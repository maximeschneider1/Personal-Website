Développement d'un CRM de gestion client (Golang, VueJS)

Dans le cadre d'un projet pédagogique avec ENGIE

Projet pédagogique développé en Golang et VueJS pour Engie My Power Dans le cadre de l’amélioration de l’expérience des conseillers de l’offre de panneaux solaires, plannification, développement d’un CRM de gestion client. J'ai également développé des algorythmes de maintenance prédictive afin de detecter des chutes de production des panneaux des clients.

Vous pouvez retrouver le code du projet sur Github
